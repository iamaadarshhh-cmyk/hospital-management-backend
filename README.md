# Hospital Management Backend

## Features
- Patient CRUD
- Doctor CRUD
- Appointment Scheduling
- Conflict Detection
- DTO Architecture
- Pagination
- Validation
- Swagger Documentation

## Tech Stack
- Spring Boot
- PostgreSQL
- JPA / Hibernate
- Lombok

## How to Run
1. Clone repo
2. Configure DB in application.properties
3. Run SpringBootApplication
4. Open Swagger UI
